## ğŸ†˜ Quick Troubleshooting

### Docker won't start?
```bash
# Check Docker is running
docker ps

# Rebuild images
docker-compose build --no-cache

# Check for port conflicts
lsof -i :3000
lsof -i :5000
```

### MongoDB connection fails?
```bash
# Verify MONGO_URI format
# Must include: mongodb+srv://user:password@...

# Check MongoDB Atlas:
# 1. IP whitelist includes your server
# 2. User has correct password
# 3. Database user has proper permissions
```

### OAuth doesn't work?
```bash
# Verify callback URLs match exactly:
# Google: https://your-backend/api/auth/google/callback
# GitHub: https://your-backend/api/auth/github/callback

# Check credentials in environment variables
# Google Client ID and Secret must be from same project
```
